# M100 Games by Kurt Dekker 

Note: All games require DUMP.CO to be loaded first. 

| File | Description                              | Bytes |
|------|------------------------------------------|------:|
| DUMP | Screen refresh utility used by ALL games |   131 |

Hit RESET button to exit games.
